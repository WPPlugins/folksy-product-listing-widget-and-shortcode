== Description ==

= WPFolksy Product Listing =

Easily add your Folksy items using a widget or shortcode into your wordpress website.

The plugin will automatically display all your latest products currently listed on your Folksy Shop page.

This is a great way to create a simple Shop page on your existing Wordpress website and offer your viewer the ability to browse your latest creations.

Each item has a direct link to their product page on Folksy. You also have the option to display a Buy button

== Installation ==

1. Backup your Wordpress database, config file, and .htaccess file
2. Upload the zip file to the `/wp-content/plugins/` directory
3. Unzip
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Visit the Folksy Listing Option screen under Settings.
6. Enter your Folksy Username in the option box.
7. You can decide to display a Buy Button or not, and choose a color.

== Frequently Asked Questions ==

= How to use the Shortcode =
* Just use [folksylisting] anywhere on your page or post.

= How to display products from a specific section =
* Use [folksylisting shop_section="NAME OF YOUR FOLKSY SECTION"]

= How to use the Widget =
* Just drop the Folksy Listing Product Widget into your sidebar.
